## 🚀 Passo a Passo

1. **Baixar todas as fotos do cliente**
2. **Renomear cada foto** conforme a tabela acima
3. **Colocar em pasta correta:**
   - Clone as pastas em: `c:\Users\guigu\OneDrive\Documentos\web\dw-company\public\images\projects\`
4. **Comprimir para web:** Usar TinyPNG ou similar (máximo 500KB por imagem)
5. **Fazer commit:**
   ```bash
   git add public/images/projects/
   git commit -m "Add project photos from current work"
   git push origin main
   ```

## 💡 Dicas

- **Tamanho máximo:** 500KB por imagem
- **Formato:** JPG (qualidade 85-90%)
- **Resolução mínima:** 1080px de largura
- **Variação:** Diferentes ângulos do mesmo projeto
