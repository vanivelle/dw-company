## 🎨 Image Naming Convention

- **Format:** `[project-type]-[description]-[number].jpg`
- **Examples:**
  - `floor-installation-light-oak-1.jpg`
  - `wooden-stairs-with-railings-1.jpg`
  - `bathroom-remodel-modern-1.jpg`
  - `interior-painting-clean-white-1.jpg`

## 📍 How to Add Images to Gallery in Code

Images are automatically picked up from these folders. To display them:

1. **In ImageGallery Component**: Images can be fetched from `/images/projects/` folder
2. **No code changes needed**: Just add images to the correct folders
3. **Responsive**: Images will automatically scale to fit different screens
